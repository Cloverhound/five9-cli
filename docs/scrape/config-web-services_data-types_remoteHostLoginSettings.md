Contains the FTP server settings for exporting recordings or reports to a remote server.

                                                    | Name | Type | Description |
| --- | --- | --- |
| hostName | string | FTP server name, which must be a fully qualified domain name or an IP address. Enter only the domain name without ftp://. |
| password | string | FTP password. |
| UserName | string | FTP user name. |
