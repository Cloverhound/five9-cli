                                                    | Parameter | Type | Description |
| --- | --- | --- |
| ivrScriptName | string | Name of script. |

#### getIvrScriptOwnershipResponse

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| return | ivrScriptOwnership | Script ownership information. |
