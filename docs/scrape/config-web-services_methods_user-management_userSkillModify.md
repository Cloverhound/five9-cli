## userSkillModify

Modifies a skill.

                                                    **Important: **To prevent the `userSkillModify` method from overloading the system, Five9 recommends that you set a sleep parameter of  at least 3 seconds between each request. Because this method affects the performance of the dialer and uses significant database resources, use this method only during off-peak periods.

#### userSkillModify

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| userSkill | userSkill | Skill to be modified. |

#### userSkillModifyResponse

Empty.
