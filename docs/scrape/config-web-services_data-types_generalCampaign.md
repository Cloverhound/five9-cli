Contains information about a campaign. These elements are returned in the [inboundCampaign](/bundle/api-config-web/page/config-web-services/data-types/inboundCampaign.htm) data type by the [getInboundCampaignResponse](/bundle/api-config-web/page/config-web-services/data-types/../methods/campaign-configuration/getInboundCampaign.htm#getInboundCampaignResponse) method.

                                                    | Name | Type | Description |
| --- | --- | --- |
| defaultIvrSchedule | ivrScriptSchedule | Configuration of an IVR script. |
| maxNumOfLines | int | Maximum number of simultaneous calls. |
