Contains the attributes of a skill.

                                                    | Name | Type | Description |
| --- | --- | --- |
| description | string | Skill description. |
| id | long | Skill ID. |
| messageOfTheDay | string | Message of the day for the skill. |
| name | string | Skill name. |
| routeVoiceMails | boolean | Whether to route voicemail messages to the skill. True: Route voicemail messages to skill. False: Do not route voicemail messages to skill. |
