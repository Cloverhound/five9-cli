## getDispositionsImportResult

Returns details about the imported dispositions.

#### getDispositionsImportResult

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| identifier | importIdentifier | Identifier of the import session, which was returned in the response to the import request. |

#### getDispositionsImportResultResponse

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| return | dispositionsImportResult | Result of disposition import operation. |
