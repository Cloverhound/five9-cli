## modifyPromptWav

Updates the settings and the audio file of a prompt.

#### modifyPromptWav

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| prompt | promptInfo | Prompt attributes. |
| wavFile | swaref | WAV file attachment in a MIME package. For more information, see the Web Services specifications. |

#### modifyPromptWavResponse

Empty.
