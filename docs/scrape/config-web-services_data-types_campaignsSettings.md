`Version 9.5`

Contains information about domain settings for running outbound campaigns.

                                                    | Name | Type | Description |
| --- | --- | --- |
| gracefulAgentStateTransitionDelay | int | Domain setting: Number of seconds between the agents’ action and the effective change of the agents’ state to Not Ready. |
| gracefulAgentStateTransitionModeEnabled | boolean | Domain setting: Whether to enable Graceful Agent State Transition. |
| priorityEnabled | boolean | Whether absolute priority is enabled. You may enable priority, ratio, both, or neither. |
| ratioEnabled | boolean | Whether priority by ratio is enabled. You may enable priority, ratio, both, or neither. |
