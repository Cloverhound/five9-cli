`Version 9.5`

Contains the number of digits that can be used for extensions assigned to users and inbound campaigns.

                                                    | Name | Type | Description |
| --- | --- | --- |
| maximalExtensionLength | int | Maximum number of digits in the user’s extension. |
| minimalExtensionLength | int | Minimum number of digits in the user’s extension. |
| minimalGeneratedExtension | string | Minimal extension number generated for a new user. |
