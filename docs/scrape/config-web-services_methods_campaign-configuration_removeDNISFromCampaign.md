## removeDNISFromCampaign

Removes a list of DNIS from a campaign. When calls are received at that number after it has been removed from a campaign,  a recording plays indicating the number is out of service.

#### []()removeDNISFromCampaign

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| campaignName | string | Name of campaign. |
| DNISList | string [0..unbounded] | List of DNIS to remove from this campaign. |

### removeDNISFromCampaignResponse

Empty.
