## deleteUser

Deletes the specified user. An exception is thrown if the user does not exist.

**Note: **When you delete a user, all associated data, including voicemails, recordings, and callbacks, will also be deleted, as indicated in the notification from the platform before the user is deleted.

#### deleteUser

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| userName | string | Name of user to be deleted. |

#### deleteUserResponse

Empty.
