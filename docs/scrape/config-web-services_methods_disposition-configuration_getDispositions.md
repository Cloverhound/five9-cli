## getDispositions

Returns the attributes of the dispositions that match a name pattern.

#### getDispositions

| Parameter | Type | Description |
| --- | --- | --- |
| dispositionNamePattern | string | Name pattern to be compared to the disposition names in the domain. |

#### getDispositionsResponse

| Parameter | Type | Description |
| --- | --- | --- |
| return | disposition[0..unbounded] | Attributes of the dispositions that match the pattern. |
