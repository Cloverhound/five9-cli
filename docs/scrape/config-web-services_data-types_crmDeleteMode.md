Contains the modes used for deleting data from the contact database.

                                                    | Name | Type | Description |
| --- | --- | --- |
| DELETE_ALL | string | Delete all records. Does not apply to single record transactions, such as when using the deleteRecordFromList method. |
| DELETE_SOLE_MATCHES | string | Delete only single matches. |
| DELETE_EXCEPT_FIRST | string | Delete all records except the first matching record. |
