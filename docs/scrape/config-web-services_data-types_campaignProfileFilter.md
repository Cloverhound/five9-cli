Contains the configurations for the campaign profile filter.

                                                    | Name | Type | Description |
| --- | --- | --- |
| crmCriteria | campaignFilterCriterion [0..unbounded] | Filter criteria for the records. |
| grouping | crmCriteriaGrouping | Configuration for the criteria of the filter group. |
| orderByFields | orderByField [0..unbounded] | Sorting order for the records in the dialer. |
