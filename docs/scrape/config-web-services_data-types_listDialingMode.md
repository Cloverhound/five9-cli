Contains the list dialing modes. These options apply to lists used with the Predictive, Power, and Progressive campaign dialing modes.

                                                    | Name | Type | Description |
| --- | --- | --- |
| VERTICAL_DIALING | string | Dialer attempts to call all numbers in a CRM record before proceeding to the next record. |
| LIST_PENETRATION | string | Dialer attempts to call all numbers in a column before proceeding to the next column. |
| EXTENDED_STRATEGY EXTENDED_STRATEGY Version 4 | string | Dialer attempts to call numbers in a list in order of importance. For example, until a contact is reached, numbers that are more important are redialed sooner and more often than those that are not. |
