Contains the possible actions to take after dialing a number.

                                                    | Name | Type | Description |
| --- | --- | --- |
| actionArgument | string | Name of prompt or IVR script to execute when a campaign-related event occurs. It is ignored when action type is DROP_CALL. |
| actionType | campaignDialingActionType | Action to take when a specific event occurs. |
| maxWaitTime | timer | Maximum waiting time for action on answering machine. |
