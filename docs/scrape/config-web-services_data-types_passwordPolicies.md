Version 2

Contains the password policies for the domain.

                                                    | Name | Type | Description |
| --- | --- | --- |
| adminLoginAttempts | int | Number of allowed login attempts for administrators. |
| enforcePasswordHistory | int | Number of passwords remembered by the system. |
| loginAttempts | int | Number of allowed login attempts for agents. |
| minCapitalCharacters | int | Minimum required number of capital letters. |
| minNumberCharacters | int | Minimum required number of alphabetical characters in the password. |
| minPasswordLength | int | Minimum length of a password. |
| minSpecialCharacters | int | Minimum required number of special characters |
| passwordExpires | int | Number of days before the expiration of the password |
