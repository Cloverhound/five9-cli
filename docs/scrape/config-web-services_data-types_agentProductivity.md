Contains the number of seconds used in reports to identify possible deviations from the expected performance of agents.

                                                    | Name | Type | Description |
| --- | --- | --- |
| longACWTime | int | Number of seconds spent performing work after a call. The time is reported if is longer than this setting. |
| longCallDuration | int | Number of seconds spent on a call. The time is reported if is longer than this setting. |
| longHoldDuration | int | Number of seconds for a call placed on hold. The time is reported if is longer than this setting. |
| longParkDuration | int | Number of seconds for a call waiting in a queue. The time is reported if is longer than this setting. |
| shortACWTime | int | Number of seconds spent performing work after a call. The time is reported if is shorter than this setting. |
| shortCallDuration | int | Number of seconds spent on a call. The time is reported if it is shorter than this setting. |
