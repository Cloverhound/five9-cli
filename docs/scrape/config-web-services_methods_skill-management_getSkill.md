## getSkill

Returns skill information: skill name, skill description, message of the day, and whether skill voicemail messages are routed to agents.

#### getSkill

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| skillName | string | Name of skill. |

#### getSkillResponse

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| return | getSkill | Skill information. |
