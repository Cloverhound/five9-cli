## createInboundCampaign

Creates an inbound campaign.

**Note: **When using the [ivrScriptSchedule](/bundle/api-config-web/page/config-web-services/methods/campaign-configuration/../../data-types/ivrScriptSchedule.htm) in this method, only the default IVR schedule can be set or modified via the API. Additionally, the name parameter of the IVR script schedule object must be excluded.

#### []()createInboundCampaign

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| campaign | inboundCampaign | Attributes of the campaign. |

#### createInboundCampaignResponse

Empty.
