## modifySkill

Modifies an existing skill. An exception is thrown if the skill already exists or if the skill attributes are invalid.

#### modifySkill

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| skill | skill | Name of skill to modify. |

#### modifySkillResponse

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| return | skillInfo | Information about the modified skill. |
