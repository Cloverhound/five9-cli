## createIVRScript

Creates an IVR script. To add details, use modifyIVRScript.

#### createIVRScript

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| name | string | Name of script. |

#### createIVRScriptResponse

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| return | ivrScriptDef | Name and description of script. |
