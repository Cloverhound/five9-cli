## createSkill

Creates a skill. An exception is thrown if the skill already exists, if the limit number of skills is reached, or if skill attributes are invalid.

#### createSkill

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| skillInfo | skillInfo | New skill information. |

#### createSkillResponse

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| return | skillInfo | New skill information. |
