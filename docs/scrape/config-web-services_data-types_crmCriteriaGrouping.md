Contains the conditions for the filter group of a campaign profile.

                                                    | Name | Type | Description |
| --- | --- | --- |
| expression | string | Expression for the group of filters if crmCriteriaGroupingType=Custom. The supported operators are AND, OR, NOT. Example: (1 AND 2 AND 3) OR (4 AND 5 AND 6 AND 7) |
| type | crmCriteriaGroupingType | Possible types of filters. |
