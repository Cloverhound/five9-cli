## addListsToCampaign

Adds dialing lists to an outbound campaign. The dialing lists must already be configured.

#### []()addListsToCampaign

                                                    | Parameter | Type | Description |
| --- | --- | --- |
| campaignName | string | Name of campaign. |
| lists | listState[0..unbounded] | Dialing lists to add to the campaign. This data type contains the campaign attributes of each list. |

#### addListsToCampaignResponse

Empty.
