Contains the attributes of a Salesforce email account.

                                                    | Name | Type | Description |
| --- | --- | --- |
| consumerKey | string | Salesforce object that is automatically generated when you save a remote access definition. |
| consumerSecret | string | Salesforce object that is automatically generated when you save a remote access definition. |
| enabled | boolean | Whether the Salesforce email integration is enabled. True: Email integration is enabled. False: Email integration is not enabled. |
| password | string | User’s password generated by Salesforce. |
| securityToken | string | Security token that you provide to your agents in addition to the password generated by Salesforce if your agents’ IP address is not in the trusted IP range configured in Salesforce for your organization. |
| userName | string | User’s name generated by Salesforce. |
